AI Immigration Case Assistant — Demo

Minimal host page for demonstrating an embedded assistant in a GD RFA–style case workflow.

Environment: Demo
Case: Training / Demo
Scope: Connected policy documents
Input: Text + Voice (if enabled)

How to use

Open the assistant from the bottom-right corner. Ask a question using text, or use voice if available.

  • 1

    Ask a case question (eligibility, required documents, steps, timelines).

  • 2

    Refine by adding specifics (relationship type, age, visa category) when prompted.

  • 3

    Verify the response against the connected knowledge sources used by the assistant.
